# Spam-Message-Classification-Project

# Overview
This project implements a simple spam message classifier using Python. 
The classifier checks for the presence of spam-related keywords in the messages a user inputs.

# Features
- Keyword-based spam detection
- Preloaded demo messages
- User input classification
- Modular code design

# Technologies Used
- Python 3
- VS Code

# How to Run
1. Install Python 3
2. Save the Python script as `spam_classifier.py`
3. Run using: `python spam_classifier.py`

# Testing
Try entering different messages to see if they are marked as SPAM or NOT SPAM. Take note of the keywords that might alter the classification.
